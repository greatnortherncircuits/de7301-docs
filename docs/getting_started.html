Getting Started¶

Congratulations on your purchase of a DE7301!

  1. Your first step to use the DE7301 is to install the hardware.

  2. After hardware installation is complete, connect it to your local network and a power source.

    Note

    The DE7301 is intended to be plugged in and left on continuously.

  3. Download the application

    • Mac Download (dmg) and drag to your Applications folder

    • Windows 64 Bit (msi) and run the installer

  4. Open the application and import your Radio Access file

  5. The application should automatically locate your radio and connect!

If the application does not connect, see troubleshooting.
